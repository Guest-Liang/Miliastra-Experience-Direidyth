# 千星奇域每周4000经验

千星奇域每周4000经验，但是从奇域收藏置顶里查找第一个关卡（即收藏但不可被搜索，只能通过收藏进入关卡游玩）   
自用
修改自https://github.com/breadgrocery/miliastra-experience-playback

## 🌟 功能介绍

- 众所周知一个奇域给20，加上成就的5x50，一局就是270，15把刚好15x270=4050
- 而你又刚好收藏了~~杀死蒂莱尔~~《打败丘丘人》这个奇域，那还说啥了

## 📋 使用方法

- 先通过调整，把奇域收藏的置顶第一个调整为~~杀死蒂莱尔~~《打败丘丘人》
- 然后自己录制一份通关脚本

  1. bgi给键鼠录制绑定一个快捷键，除wasd和f，以及鼠标左键以外都可以
  2. 进奇域，等待完全加载，鼠标别动，按下快捷键观察日志窗口，等待录制开始
  3. 录制开始后，按w往前走，鼠标可以微调使得面向丘丘人，门口的坎不用跳，能走上去，等走上去之后多次点按f并按鼠标左键攻击，确保击败丘丘人以后能捡到钥匙
  4. 能通关了就停止录制，可以先自己加载进奇域，然后播放一下看看能不能通关，不能的话重复第三步多试试
  5. 最后复制键鼠录制文件到脚本目录的assets/playbacks文件夹下，把文件名填到js脚本配置里去，同时记得检查删除存档的名字是不是能对应上

## 🛠️ 脚本配置

| 配置项    | 名称           | 描述           | 默认值 |
| -------- | -------------- | -------------- | ------ |
| playbacks | 通关回放文件池  | 逗号分隔，随机抽取，自行录制拷贝到assets/playbacks  | 19910311143.json  |
| deleteStageSaveKeyword | 删除关卡存档关键字  | 关卡存档视图中的[关卡]列  | 19910311143  |
| thisAttempts  | 指定通关次数  | 0表示自动判断  | 15   |

## ❗ 注意事项

- 请确保游戏窗口分辨率比例为 `16:9`，否则可能影响脚本正常运行。
- 建议提前设置好快捷键，便于在需要时快速暂停或终止脚本。
